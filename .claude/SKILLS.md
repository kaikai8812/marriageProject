# Marriage Project スキル一覧

## /add-section
LPに新しいセクションを追加する。ヒーロー、挨拶文、ギャラリー、式場案内、出欠確認など、追加したいセクションの種類を指定して使う。

## /preview
ローカル開発サーバーを起動してLPをプレビューする。`npx serve src` を実行し、ブラウザで確認できる状態にする。

## /deploy
現在の変更をコミットし、GitHub Pages にデプロイする。変更内容の確認 → コミット → push → デプロイ確認まで一連の流れを実行する。

## /responsive
LPのレスポンシブ対応を確認・修正する。スマホ(375px)、タブレット(768px)、デスクトップ(1024px+)の各画面幅での表示を最適化する。

## /optimize-image
LP内の画像を最適化する。引数に画像パスまたは「all」を指定する。ファイルサイズと解像度を適切なレベルに調整する。
