# Marriage Project - ウェディング招待LP

## プロジェクト概要
結婚式の招待LP（ランディングページ）。リンクを知っている人が誰でもアクセスできるWebページ。

## 公開URL
https://kaikai8812.github.io/marriageProject/

## 技術スタック
- **静的サイト**（ビルドツール不要）
- HTML / CSS / JavaScript のみ
- `src/` 配下のファイルがそのままデプロイされる

## ファイル構成
```
src/
├── index.html    ... メインページ
├── style.css     ... スタイルシート
├── script.js     ... JavaScript（必要に応じて）
└── images/       ... 画像ファイル置き場
```

## デプロイ
- `main` ブランチにpushすると GitHub Actions で自動デプロイ（約15〜30秒）
- ビルド不要。`src/` の中身がそのまま公開される

## ローカルプレビュー
```bash
npx serve src
```
で `http://localhost:3000` にてプレビュー可能。

## コーディングルール
- モバイルファースト（スマホでの閲覧が主）
- 画像は `src/images/` に配置し、相対パスで参照する
- CSSは `style.css` に集約する（ファイルが大きくなったらセクションごとにコメントで区切る）
- JavaScriptは `script.js` に集約する
- 外部ライブラリはCDN経由で読み込む（npm不要）
- コメントは日本語で記述する

## デザイン方針
- ウェディング招待状にふさわしい上品で落ち着いたデザイン
- スマートフォンでの表示を最優先
- 画像の読み込みパフォーマンスに配慮する（適切なサイズ・圧縮）
