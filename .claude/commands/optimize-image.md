# 画像最適化

LP内で使用する画像を最適化する。

## 引数
$ARGUMENTS ... 最適化対象の画像パスまたは「all」で全画像

## 手順

1. 対象の画像を特定する:
   - 引数が「all」の場合は `src/images/` 内の全画像
   - それ以外は指定されたパスの画像
2. 各画像について以下を確認する:
   - ファイルサイズ（目安: 写真は200KB以下、アイコンは50KB以下）
   - 解像度（LPで必要なサイズ以上に大きくないか）
3. 最適化が必要な画像に対して `sips` コマンド（macOS標準）でリサイズする:
   ```bash
   sips --resampleWidth 1200 画像パス
   ```
4. 最適化結果をユーザーに報告する（ファイル名、変更前後のサイズ）

## ルール

- 元画像は上書きする（Git管理されているため復元可能）
- ヒーロー画像など全幅表示の画像: 幅1200px以下
- ギャラリーのサムネイル: 幅600px以下
- JPEG品質は適切なレベルを保つ
- WebP形式への変換が可能であれば提案する
