# セクション追加

LPに新しいセクションを追加する。

## 手順

1. `src/index.html` を読み、現在のセクション構成を把握する
2. ユーザーに追加したいセクションの種類を確認する（以下は例）:
   - ヒーロー（メインビジュアル）
   - 挨拶文
   - フォトギャラリー
   - 式場案内・地図
   - 出欠確認（RSVP）
   - カウントダウンタイマー
   - プロフィール紹介
   - タイムライン（当日のスケジュール）
   - SNS共有ボタン
   - その他（自由記述）
3. セクションのHTMLを `src/index.html` の適切な位置に追加する
4. 対応するCSSを `src/style.css` に追加する
5. 必要ならJavaScriptを `src/script.js` に追加する

## ルール

- 既存のセクションとデザインの一貫性を保つ
- セクションは `<section id="セクション名">` で囲む
- CSSにはセクションごとにコメントで区切りを入れる（例: `/* ===== ギャラリー ===== */`）
- モバイルファーストで実装する
- 画像を使う場合は `src/images/` に配置し、適切な alt テキストを付ける
