# レスポンシブ確認・修正

LPのレスポンシブ対応を確認し、必要に応じて修正する。

## 手順

1. `src/style.css` を読み、現在のレスポンシブ対応状況を把握する
2. 以下の画面幅を想定して確認する:
   - **スマートフォン**: 375px（iPhone SE / 標準的なスマホ）
   - **タブレット**: 768px（iPad）
   - **デスクトップ**: 1024px 以上
3. 問題がある箇所を特定し、メディアクエリを追加・修正する
4. 修正した内容をユーザーに報告する

## ルール

- モバイルファースト（デフォルトのCSSはスマホ向け、大画面はメディアクエリで対応）
- ブレークポイント:
  ```css
  /* タブレット以上 */
  @media (min-width: 768px) { }
  /* デスクトップ以上 */
  @media (min-width: 1024px) { }
  ```
- フォントサイズ、余白、画像サイズが各画面幅で適切か確認する
- 横スクロールが発生しないことを確認する
